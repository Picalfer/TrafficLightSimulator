# TrafficLightSimulator

[![Kotlin](https://img.shields.io/badge/Kotlin-1.8+-blue.svg?logo=kotlin)](https://kotlinlang.org)
[![Platform](https://img.shields.io/badge/Platform-Android-green.svg?logo=android)](https://www.android.com/)

Тестовое приложение-симулятор светофора с автоматическим переключением цветов.

## О проекте

Приложение имитирует работу светофора с циклическим переключением цветов (красный-желтый-зеленый). Демонстрирует работу с таймерами и управление состоянием UI.

**Основной функционал:**
- Автоматическое переключение цветов светофора
- Кнопка старт/стоп для управления симуляцией
- Визуальное отображение всех состояний светофора
- Сброс к исходному состоянию при остановке

## Технологии

* Kotlin
* Timer
* ImageView
* ImageButton
* State management
